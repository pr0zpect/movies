# The Movie Mania

🎯 Purpose

The goal of this project is to:

Build a dynamic and interactive frontend
Implement features like search, filtering, and state management
Create a real-world usable application

🌐 APIs Used
1. OMDb API
🔗 https://www.omdbapi.com/
Used for fetching movie data like:
Title
Poster
Year
Ratings

✨ Features
✅ Core Features
🔍 Live Movie Search
Fetch results from API as the user types or clicks search
Display movies in real-time

📌 Sticky Watchlist Sidebar
Displays saved movies
Always visible while browsing
🌟 Optional Feature
🎲 Random Movie Night
Picks a random movie from the watchlist
Helps users decide what to watch

🎨 UI/UX Design
Dark theme inspired by Disney+ / HBO Max
Clean and modern layout
Responsive design
Smooth hover and transition effects

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
API Handling: Fetch API / Axios
State Management: Local Storage (for watchlist persistence)
Version Control: Git & GitHub
