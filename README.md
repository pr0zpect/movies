# The Movie Mania

🎯 Purpose

The goal of this project is to build a dynamic and interactive frontend application for discovering and managing a list of favorite movies. It implements features like search, sorting, filtering, favorites management, and theme toggling.

🌐 APIs Used
1. OMDb API
🔗 https://www.omdbapi.com/
Used for fetching movie data like:
- Title
- Poster
- Year

✨ Features
✅ Core Features
- **Movie Search:** Fetch results from the OMDb API by searching for a movie title.
- **Default Movie List:** Initially loads a curated list of classic movies (e.g., Inception, Interstellar, The Matrix).
- **Favorites System:** Mark movies as your favorites and easily toggle a view to show only your favorite movies.
- **Sorting:** Sort the displayed movies alphabetically by "Title" or chronologically by "Year".
- **Filtering:** Filter the displayed movies by specifying a minimum release "Year".
- **Theme Toggle:** Switch the UI between Light Mode and Dark Mode to suit your preference.

🎨 UI/UX Design
- Clean and modern layout.
- Responsive design.
- Interactive buttons and input feedback.

🛠️ Technologies Used
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **API Handling:** Fetch API
- **State Management:** In-memory variables and DOM manipulation
- **Version Control:** Git & GitHub
